# 系统文档

## 性能指标

我们的系统可以达到每秒 10 万消息的处理能力。

延迟控制在 10ms 以内。

## 技术栈

后端采用 Java 语言开发。

使用 Spring Boot 框架。

数据存储使用 MySQL。

缓存使用 Redis。

## 部运维

系统支持 Docker 部署。

也支持 Kubernetes 部署。

## 监控告警

使用 Prometheus 进行监控。

配置告警规则。

## 核心功能

消息的发送和接收。

消息的持久化。

消息的过滤和路由。

## 系统概述

这是一个分布式消息队列系统。

用于解决系统间的异步通信问题。

## 架构设计

采用微服务架构。

包含消息代理、存储引擎等组件。
