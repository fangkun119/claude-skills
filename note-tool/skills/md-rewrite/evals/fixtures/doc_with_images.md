# 消息队列架构

## 系统架构图

下面的架构图展示了系统的整体结构：

![系统架构](./images/architecture.png)

从图中可以看到，系统包含三个核心组件：消息代理、存储引擎和管理接口。

## 消息流转流程

消息在系统中的流转过程如下图所示：

![[./images/message_flow.png]]

消息生产者将消息发送到消息代理，消息代理根据路由规则将消息存储到存储引擎，消息消费者从存储引擎中拉取消息。

## 组件交互时序图

各组件之间的交互时序：

![组件交互](./images/sequence.png)

首先是连接建立，然后是消息发送，最后是消息确认。
