## 一、章节核心主题与观点

### 1.1 核心主题
本章从中金大类资产配置团队的专业视角，阐述流动性作为左右全球资产走势的核心因素，提出"外汇脉冲"这一新的流动性分析框架，并基于对中国和海外流动性的分析，给出具体的全球资产配置建议。

### 1.2 核心观点
1. **流动性是左右全球资产的核心因素**
   - 在高预期、高估值、高拥挤度的市场环境中，流动性将决定资产走势
   - 流动性宽裕可能是牛市持续的燃料，流动性收紧可能是重大风险的触发器

2. **中国流动性结构已发生根本变化**
   - 过去十几年：信用驱动（信贷脉冲、社融增速等指标有效）
   - 2026年后：非信用渠道驱动（央行扩表、财政投放、外汇流入）
   - 新框架：外汇脉冲成为更有效的流动性追踪指标

3. **海外流动性面临三重风险但长期向好**
   - 短期：地缘、通胀、政策三重风险叠加，由降息预期转为加息预期
   - 长期（3-6个月）：三重风险均可能缓解，海外流动性有望重新宽松

4. **具体资产配置建议**
   - 超配：中国股票、黄金
   - 标配：美股、美债、商品（可逢低布局）
   - 低配：中债（比价角度）
