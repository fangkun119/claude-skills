## 一、核心主题与观点

### 1.1 主题定位
本章聚焦**分化中共振**框架下的国内经济分析，从"名义增长回升"这一主线出发，回答三个核心问题：
- 名义增长回升的结构性去向（外需超预期）
- 内需持续疲软的深层原因（去杠杆模式差异）
- 房地产企稳的关键观察信号（土地出让收入）

### 1.2 核心观点
1. **名义增长如期而至，但结构集中在外需**
   - 年初预期的三个反转（通胀、房价、消费）只有出口明显超预期
   - AI对出口增速的贡献超过1/3，且下半年能见度较高

2. **预期模式存在系统性偏差**
   - 过去几年市场持续低估中国出口、高估内需
   - 需要回归基本面分析，克服心理偏见

3. **内需疲软源于去杠杆模式差异**
   - 中国缺乏债务重组，居民债务负担持续存在
   - 内需复苏的兑现时间因结构性因素而延后

4. **房地产企稳需观察关键信号**
   - 今年是"最健康的一次"地产小阳春（二手房挂牌量平稳）
   - 真正企稳需验证：土地出让收入回升且房价稳定

## 三、保留的关键数据和要点

### 3.1 核心数据（必须保留）

#### 名义增长与出口数据
- **年初出口增速预期**：6-8%（市场当时认为偏乐观）
- **当前出口预测**：下半年增速10%以上，全年两位数增长
- **AI对出口贡献**：超过1/3（直接产业相关性测算）
- **PPI同比拆分结论**：供需两端拖累都在减少，主要来自供给端

#### 预期偏差数据
- **23年出口预测与实际**：差距不大
- **24年、25年及今年一季度出口**：市场一致预期大幅低估
- **内需定义**：社会消费品零售总额 + 固定资产投资总额
- **内需实际与预测缺口**：23年以来持续扩大（灰色线vs红色线）

#### 房地产数据
- **历史规律观察**：房价企稳通常伴随土地出让收入增速见底反弹
- **当前土地出让收入**：截至4月份，同比跌幅仍在扩大
- **二手房挂牌量**：今年地产小阳春期间大体平稳

### 3.2 关键观点要点（必须保留）

#### 关于外需
- 名义增长回升的主线没有变，但集中在出口领域
- AI对出口的拉动短期内不会明显下降（订单能见度高）
- 外需强劲的可持续性：基于上市公司和海外资本开支预测，短期争议较小

#### 关于内需
- 市场对中国出口容易低估、对内需求高估（过去几年回溯）
- 内需疲软根源：去杠杆模式差异（美国靠债务重组，中国缺乏债务重组）
- 居民债务负担持续存在，对需求压制一直存在
- 内需会有底，但复苏兑现时间因结构性因素而延后

#### 关于房地产
- 今年是"最健康的一次"地产小阳春（二手房挂牌量平稳，供需结构相对健康）
- 真正企稳的验证标准：土地出让收入回升 + 房价稳定
- 当前仍存在供需矛盾：新房供给过快会压制房价
- 土地出让收入与税收的互补关系还需观察

## 五、专业术语与表达规范

### 5.1 宏观经济术语
- **名义增长**：指包含价格因素的经济增长，区别于实际增长
- **PPI**：生产者物价指数（Producer Price Index）
- **去杠杆**：降低债务收入比的过程，可通过债务重组或增长稀释实现
- **债务重组**：通过减少债务本金或利息来降低债务负担
- **内需**：国内需求，由消费（社会零售品总额）和投资（固定资产投资）组成
- **土地出让收入**：地方政府通过出让国有土地使用权获得的收入

### 5.2 数据与图表说明
- 文中提到多个图表（如PPI同比拆分图、出口拉动拆分图、预期偏差回溯图），需在改写时：
  - 保留图表引用（如"如图X所示"）
  - 说明图表含义（如"图中灰色线代表实际值，红色线代表预测值"）
  - 数据来源标注（如Bloomberg一致预期、中金测算）

### 5.3 表达规范
- 使用陈述句，避免疑问句和反问句
- 使用专业词汇，避免口语化比喻
- 数据需精确（保留原文的百分比、增速等）
- 逻辑连接词：因此、然而、此外、基于此、综合来看
