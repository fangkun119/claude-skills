## 一、核心主题与观点

### 1.1 章节定位
本章是A股市场的专场分析，承接第三章的国内经济展望，为第五章的资产配置建议提供A股层面的具体指引。

### 1.2 核心观点
1. **趋势判断**: 924行情延续，中期趋势未改，短期调整不改变中期向好趋势
2. **节奏把握**: 市场节奏从"前升后稳"转向"稳好于快"，当前处于短期调整后期
3. **配置思路**: 从"谋新温差收敛"调整为"胜孤择优"，注重基本面和业绩确定性

### 1.3 关键数据点
- 上证指数开年微负收益，最高涨超7%，目前已回吐涨幅
- 两次风险提示：开年17连阳第17天、5月13日万德全A创新高
- 一次机会提示：3月23日跌破关键点位后提示买入机会
- 盈利预测：非金融上调至接近10%，金融上调至3%，全年预期6%（过去5年最好水平）
- 估值分化：小微盘估值偏高，成长股内部分化明显（储能/光模块/PCB vs 算力芯片/半导体设备）
