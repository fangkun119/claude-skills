## 语言风格规范

### 词汇选择
- 使用正式书面语：避免"那"、"呢"、"啊"等口语词
- 保留专业术语：K型分化、硅基与碳基、科技泡沫、市场拐点
- 精简冗余表达：去除重复和冗长的描述

### 句式结构
- 使用陈述句为主，避免疑问句的口语化表达
- 采用主动语态，增强表达力度
- 适当使用排比句式增强节奏感

### 逻辑连接
- 使用"首先"、"其次"、"此外"等正式连接词
- 避免"那"、"所以"、"然后"等口语连接词
- 保持段落间的逻辑递进关系
